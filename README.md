## 💼 Corporate Income Tax Fraud Detection Dashboard

🚀 An intelligent fraud detection dashboard built using Python, Streamlit, and Machine Learning to analyze corporate financial data and identify potential tax fraud activities.


# ✨ Features

✅ Secure Login Authentication  
📂 Upload Corporate Financial CSV Files  
🧹 Data Cleaning & Preprocessing  
🤖 Fraud Prediction using ML Models  
📊 Interactive Dashboard & Visualizations  
📄 PDF & CSV Report Export  
📈 Feature Importance Analysis  
🔍 Fraud Distribution Insights  

# 🛠️ Technologies Used

- 🐍 Python
- ⚡ Streamlit
- 📑 Pandas & NumPy
- 🤖 Scikit-learn
- 🚀 XGBoost
- 📊 Plotly
- 📉 Matplotlib & Seaborn
- 📄 ReportLab


# 🤖 Machine Learning Models

- 🌲 Random Forest
- 📐 Logistic Regression
- 🚀 XGBoost


# 📊 Dashboard Functionalities

✔️ Fraud Detection Analytics  
✔️ ROC Curve & Confusion Matrix  
✔️ Industry & Location-based Analysis  
✔️ KPI Metrics  
✔️ Dynamic Filtering System
